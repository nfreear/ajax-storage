# ajax-storage

`localStorage` and `sessionStorage` caching seamlessly built onto jQuery Ajax HTTP requests. For the browser.

* [@nfreear/ajax-storage][]


## Usage:

```js
  $.ajax_storage({
    url: 'http://...',
    success: function (data) {
      // Do something with the data.
    }
  });
```
